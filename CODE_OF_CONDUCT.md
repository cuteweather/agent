# Code of Conduct

This repository aims to be welcoming, practical, and respectful.

## Expected Behavior

- Be respectful in language and tone.
- Give technical feedback that is specific and constructive.
- Assume good intent, but stay honest about problems and risks.
- Respect privacy, security, and licensing boundaries.
- Help keep discussions focused on improving the repository.

## Unacceptable Behavior

- Harassment, intimidation, or personal attacks
- Discriminatory or hateful language
- Deliberate spam, trolling, or derailing discussions
- Sharing private credentials, personal data, or other sensitive material

## Maintainer Responsibilities

Maintainers may edit, hide, lock, or remove discussions and contributions that violate these guidelines.

## Reporting

If you run into behavior that makes collaboration unsafe or unproductive, open a private channel with the maintainer if possible instead of escalating publicly.
